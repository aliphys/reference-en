= sqrt(x)


// OVERVIEW SECTION STARTS
Calculates the square root of a number.
[#overview]
--

[float]
=== Description
[%hardbreaks]


[float]
=== Syntax
`sqrt(x)`


[float]
=== Parameters
`x`: the number. Allowed data types: any data type.


[float]
=== Returns
The number's square root. Data type: `double`.
